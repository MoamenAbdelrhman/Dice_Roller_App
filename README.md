# 🎲 Dice Roller App

A simple Android application built with **Kotlin** that simulates rolling dice.

When the user presses the **Roll** button, the app generates random numbers and displays the corresponding dice images on the screen.

---

## 📱 Features
- Roll two dice randomly
- Display dice images based on the generated numbers
- Simple and clean user interface
- Built using ConstraintLayout

---

## 🛠️ Technologies Used
- Kotlin
- Android Studio
- ConstraintLayout
- ImageView
- Button

---

## ▶️ How It Works
1. The user presses the **Roll** button.
2. The app generates a random number between **1 and 6** for each dice.
3. The corresponding dice image is displayed on the screen.
